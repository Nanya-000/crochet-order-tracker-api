# Crochet Order Tracker API

## Project Overview
The Crochet Order Tracker API is a backend application built to help small crochet businesses track customer orders and their statuses. It allows users to create and view orders through simple API endpoints.

This project was developed as part of the ALX Backend Web Development Capstone Project.

## Tech Stack
- Python
- Flask

## Features
- API status endpoint
- Create new crochet orders
- View all existing orders
- Simple in-memory data handling (can be extended to a database)

## API Endpoints
- GET / → Check API status
- GET /orders → Retrieve all orders
- POST /orders → Create a new order
